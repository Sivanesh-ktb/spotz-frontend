Import Bookings

Import bookings from a CSV file.

Upload File
  • Create a blank first column
  • Fields required:
    • facility
    • space
    • date
    • start (time)
    • end (time)
    • eventname
    • attendance (10, 20, 50, 100, 200, 300, 400, 500, 1000, 3000, 3001)
    • customer
    • email
    • eventtype (sport)
  • Time format: hh:mm am
  • Items should be limited to 300 at a time (rework underway)
  • Processing is 0.3s/line, and it should complete before a timeout of 60s
