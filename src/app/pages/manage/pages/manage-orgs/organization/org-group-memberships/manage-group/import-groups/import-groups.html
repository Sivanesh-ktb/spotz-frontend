Import

View instructions on import file formatting for each of the following:

Groups and members optional

Upload a CSV file that has the following fields. Use the Header Text underlined in the first row of each column

Field / Header Text Type Description
Group name (required) Text String Name the group so others can categorize themselves (e.g. Staff, Coach, etc.)
level (Priority) Text String or Number
  • "First" or 4
  • "Second" or 3
  • "Third" or 2
  • "Fourth" or 1
  • "Everybody" or 0 -- Default
 Levels have access to any listings at or below the level. i.e. Second priority groups can rent Second, Third, Fourth or Public spaces listings.
Requiredlead Time Number (0-10)
Default 2		 Number of days (x 24hrs) that the group needs to book a space.
discount Percentage Number (0-100)
Default 0		 This is the percentage discounted from a space listing. e.g. 5% discount would save a group $5 on a $100 rental, and a 100% discount is a $0 rental.
instant Approval True / False
Default False		 If True, members of this group do not require approval to use the space.
tax Exempt True / False
Default False		 If True, members of this group will not be charged Sales Tax.
private / Invite Only True / False
Default True		 If True, members must be invited to join the group (no self-application).
member.email (required)
member.firstName (optional)
member.lastName (optional) 		Text String For each user that you would like to add to the group, you must at least include the group name in the row. Email is Required
